# Desafio_Tecnico
Projeto de Inteligência Artificial voltado à identificação precoce de estudantes em risco de evasão no ensino superior, com foco explícito em avaliação e mitigação de vieses socioeconômicos. O trabalho utiliza métricas de fairness e estratégias de pós-processamento para garantir equidade no apoio à tomada de decisão institucional.
